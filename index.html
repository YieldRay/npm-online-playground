在控制台即时加载npm包

主要给出了两个函数：require() 和 appendPackageToHead() ，函数已挂载至window

require函数仅能导入完全兼容RequireJS的包

例如，React默认以CommonJS形式导出，直接 const react = require("react") 即可

或者，直接从url加载 const react = require("https://unpkg.com/react@18/umd/react.production.min.js")

Vue3默认以global形式导出，使用 await appendPackageToHead("vue") 之后就可以使用 Vue 对象了

Vue2： const Vue = require("vue@2")

兼容iife形式的包也可以使用 await appendPackageToHead()

不支持 es6 import/export

移动端可以导入 vconsole 或 eruda

直接在本页面加上请求参数 eruda 或 vconsole 即可启用

例如：http://example.net/index.html?eruda
